# Chat

Normas de Instalacion


1 cd Chat

2 composer install

3 cp .env.example .env

4 Configurar el .env

5 php artisan key:generate

6 Crear en el Mysql la base de datos, con el nombre proyectoChat

7 php artisan migrate

8 php artisan db:seed
